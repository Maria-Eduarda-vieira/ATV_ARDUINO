Em um primeiro momento, criei o circuito baseado no exemplo que o professor ensinou em sala de aula e acendi apenas um led, apresentado no arquivo ATV ARDUINO.1. Depois, aprimorei o circuito incluindo mais dois leds e fazendo a programação necessária para fazer uma sequência de iluminação. Ainda, identifiquei a finalidade de cada passo do código.
